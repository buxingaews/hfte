　　《幽美小作品》幽美小作品（一）：合奏那轻脆的铃响，那秀美的穗子，只为才子那动听的笑靥。长安瑞雪后，那宫阙重楼里，元宵灯海美如昼。一手莲灯，一个理想，一柄萤扇，一丝心凉，一壶琼浆，一滴泪水。轻迈脚步，游走人群，牵着独立，看榴莲果花瘦。花灯里的凉爽，过眼云烟的美，欢声笑语，早已变泪空流。湖面上的小舟，烟花凋谢的哀，金碧辉煌，十足不过假装。香腮胜雪不过为了掩饰苦楚泪水，鬓影华衣然而为了湮没累累创痕，冰花芙蓉玉晶莹，霓裳羽衣舞曼柔。独立的人能否会在功夫里径自羸弱？孔明灯的光，能否长久闪烁，月球的柔光，能否长久凉爽？月色下的才子，抚琴，赞美，琴声瑟瑟，笛音箫箫，谁在伴乐，本人人合奏，片刻琴声，片刻笛音，独立的乐声，梦想着完备。曲终人散后，大概还能再重头，重头？还好吗重头幽美小作品（二）：念书，我的痛快念书是一件痛快的工作。对于爱念书的人一但读上就让人骑虎难下，没辙停止念书，想让人读到天南地北。有人说：人生最深最宽厚的痛快，即是静观天下与人生，渐渐品位出它的和谐与美。静下心来，翻开书籍，那些堆积的墨香一点点溢满空间，那些尘封的痛快一点点翻开。念书真的很痛快!到了初级中学，咱们有七门作业，但是我最爱好读的作业是汗青和语文：我爱好读汗青是由于我想领会咱们华夏建国五千年此后的灿烂和羞耻。我要以咱们国度的灿烂引觉得豪，将咱们国家的羞耻铭刻在意;读语文是由于我想领会咱们华夏的文明普通是如许的丰富，唐诗宋词是多么的让人沉醉，咱们华夏有几何超过于寰球前线的文明。歌德往日说过：读一本好书，即是和一位高贵的人说话。是的，读好书，就像缔交了良师良友，使我收获颇丰;读好书，就像洗浴在和缓的东风里，使我赏心悦目;读好书，就像找到了痛快之源，使我领会到痛快的真理!书，是我超过的门路;书，是我翻开聪慧大门的钥匙;书，是我灵感和创造的来源;书，是我进步的马达;书，是我生存的目标盘;书，是我生存中不行缺乏的一局部。书，就像一台吸尘器，取消了人们精神中的缺点;书，就像我的良知，伴咱们流过生存中的沧桑。我爱书，更爱念书，书是我的痛快之源，将使我终身受用不尽。书，我的痛快之源，将伴我终身幽美小作品（三）：人命人命如画，有浓墨倾泻，也有淡笔轻描；人命如歌，有轻吟浅唱，也有黄钟大吕。君不见李太白、杜工部之终身，浓墨倾泻，铸就人命之宏大；裴多菲、雪莱之终身，淡笔轻描终写尽人命之真理。君不见轻音浅唱之陶渊明采菊东篱下，悠然见南山，斗志昂扬之谭嗣同我自横刀向天笑，去留肝胆两昆仑。人命是山，咱们没辙预估它的长度，咱们只好探求它的莫大。人命是路，它是由一块块不起眼的沙石构成的，而不是铺金布银，平平常淡中尽显人命之完备。人命是一叶扁舟，飞翔于茫茫桑田之中，惟有体验狂风雨的浸礼，才会迎来来日肩上初升的太阳。幽美小作品（四）：幼年回顾中，有一个很大的果木园，园子的范围，密密地怒放着一圈金色色的葵花。一个纤细的小女孩，坐在土埂上，仰着小脸，对着葵花如痴如醉地浅笑。葵花顽固而暖人的浅笑，映刻在我幼稚的心中，纵然雪窖冰天的冬天到来之时，也似乎能体验那灿然的和缓。秋天，葵花接踵老练，一颗一颗剥着，吃着，那潮湿而芳香的丝丝甜味，缭绕在唇边，浸透到内心。年纪渐长，葵花怒放的相貌在意中渐渐淡出，有时机看法了更多朵儿，但再没嗑那种新鲜的湿湿的葵花籽的时机了。天下面凡事都要开销价格，哪怕只是是一种爱好。由于嗑瓜子，我自觉得很美丽的两颗门牙，悲惨都转成了半颗。一位作者死于暗疾，网上所以而打开了庄重的祝贺震动，很多很多秀美的朵儿，我不假思索献上去的，竟是葵花。注意着屏幕上那一望无穷的葵花，思路锁定在幼年时的谁人大果木园里，似乎看见，盛开着的那一圈金色色的葵花。寰球上再也没有第二种朵儿，对太阳的爱，如许迷恋如许顽固，如许安静而潜心，从开放的发端到中断。寝室窗前有块小花圃，我种下了很多葵花。花开时节，坐在宽大的飘窗前凝视，似乎又回到了高枕无忧的幼年。幽美小作品（五）：精力精力是一座丰碑，是一块基石，是进步路途上的灯塔，是绚烂人生的标尺。有了矜持，学富五车、学富五车的孟子渐渐吟出三中国人民银行，必有我师，虚怀若谷，终成一代儒学巨匠；有了宽容，野心勃勃、斗志昂扬的齐桓公唾弃新仇旧恨，知人善任，功效千秋伟业，终成一代霸主；有了坚忍，宁静忧伤、凄凉郁愤的苏武一直不扔掉手中的汉使族节，心系故国，终成千秋爱国典型；有了贡献，慈爱淳厚、卑鄙无闻的徐虎才不妨十年如一日，情洒人民，和缓万家，终成现在的活雷锋。精力，穿梭汗青地道，趟过汗青长河，将长久撑起人生陡峭宏伟的高楼。幽美小作品（六）：墨夜夜，如墨般滴在天际的画布上，浸透、晕开，用画笔轻点几下，便有了星斗，再描上几下，温柔的月色将射进你的心房。萤火虫的光足以媲美点点繁星，却要和缓几分。蛐蛐儿拿出小中提琴，与和风协作，吹奏出一曲幽美的圆圆舞曲。溪水渐渐地流着，像功夫的保送带，一点点流
